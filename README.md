A highly interactive, creative developer portfolio website designed with a **Vintage Botanical Herbarium** logbook theme. Featuring realistic paper textures, hand-drawn vector illustrations, and immersive, tactile widgets that transform standard web interactions into a sensory experience.

---

## 🌿 Key Features

1. **3D Fluttering Butterfly** — A terracotta-winged SVG butterfly hovering near your name card that flutters its wings in 3D (`rotateY` keyframes) and flies away gracefully to the top-right corner when hovered.
2. **Interactive Wooden Flower Press** — A wood-textured flower press widget with corner wing-nuts (screws). Clicking on the wing-nuts plays a spin animation and cycles through three layered cards (*The Naturalist*, *Core Traits*, *Field Notes*) that slide out. You can also click directly on the paper tabs sticking out.
3. **Concentric Tree Growth Rings Timeline** — An interactive wood slice trunk displaying rings for years 2023 to 2026. Hovering/clicking on growth rings highlights the ring and updates the work milestone descriptions on a side-pinned clipboard notepad.
4. **Seed Packets Toolkit** — Vintage seed packets housing skill tags that slide up when hovered, complete with custom seed seedling vector SVGs.
5. **Archival Specimen Sheets** — Project cards designed as archival specimen sheets with detailed double borders, cardboard tags, and transparent glass vials containing dried flowers.
6. **Pinned Envelope Contact Card** — Invitation envelope layout with a paper flap, pinned with a pressed red rose SVG secured with a metallic safety pin.
7. **Peeking Cat Silhouette** — Symmetrical black cat silhouette nestled in a bed of vector grasses and fern branches. The golden eyes contain pupils that track the cursor dynamically.

## 🛠️ Stack & Architecture

- **Core**: Vanilla HTML5, CSS3 Variables, Custom SVGs, and Interactive JavaScript
- **Build System**: Vite 5 (For fast packaging, live reloading, and assets optimizations)
- **Palette**:
  - Linen Background: `#faf7f2`
  - Leaf Green: `#3d5a45`
  - Terracotta Clay: `#c67d6c`
  - Peach Blush Flap: `#f9e5db`
- **Fonts**:
  - `DM Serif Display` (Elegant headings)
  - `Caveat` & `Sacramento` (Handwritten ink fountain-pen scripts)
  - `Outfit` (Legible body text)

## ⚡ Setup & Run

### 1. Installation

Install Node modules and project dependencies:

```bash
npm install
```

### 2. Local Development

Run the local Vite development server on `http://localhost:3000`:

```bash
npm run dev
```

### 3. Build Production Bundle

Compile and bundle assets for hosting:

```bash
npm run build
```
